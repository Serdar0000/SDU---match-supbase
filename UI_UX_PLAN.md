# SDU Match - UI/UX Refactoring Plan

**Дата создания:** 25 февраля 2026  
**Цель:** Масштабная переработка UI/UX для улучшения пользовательского опыта

---

## 📊 Текущее состояние

### Существующие страницы
1. **Login Page** - Авторизация/Регистрация
2. **Onboarding Page** - Первичная настройка профиля
3. **Swipe Page** - Главная страница со свайпами
4. **Profile Page** - Профиль текущего пользователя
5. **Matches Page** - Список матчей и чаты
6. **Chat Page** - Индивидуальный чат
7. **Settings Page** - Настройки приложения
8. **Edit Profile Page** - Редактирование профиля

### Текущая навигация
- Bottom Navigation Bar с 3 вкладками:
  - Profile (Профиль)
  - Home (Свайпы)
  - Matches (Матчи/Чаты)
- GoRouter для управления роутингом
- StatefulShellRoute для сохранения состояния табов

### Технологический стек
- Flutter + Dart
- GoRouter для навигации
- Supabase для backend
- GetIt для DI
- Freezed для моделей
- flutter_card_swiper для свайп механики

## 🎯 Приоритетные направления улучшений

### 1. 🎨 Визуальный дизайн

#### 1.1 Цветовая схема
**Текущие проблемы:**
- Стандартные цвета Material Design
- Недостаточно фирменного стиля SDU

**Предложения:**
- [ ] Создать фирменную палитру SDU Match
  - Primary: Градиент от синего к фиолетовому (университетские цвета)
  - Accent: Яркий оранжевый/коралловый для действий
  - Background: Тёмный режим с мягкими градиентами
- [ ] Добавить плавные переходы между экранами
- [ ] Создать консистентную систему теней и радиусов

#### 1.2 Типографика
- [ ] Внедрить иерархию шрифтов (настроить AppTheme)
- [ ] Использовать более современный шрифт (например, Inter или SF Pro)
- [ ] Адаптивные размеры для разных экранов

#### 1.3 Иконография
- [ ] Заменить стандартные иконки на кастомные
- [ ] Добавить анимированные иконки для ключевых действий
- [ ] Единый стиль (outline или filled)

### 6. 🚀 Onboarding Experience

**Текущие проблемы:**
- Длинная форма на одной странице
- Может быть overwhelming для новых пользователей

**Улучшения:**
- [ ] **Разбить на шаги:**
  1. Добавить фото + Имя
  2. Основная информация (возраст, факультет, курс)
  3. О себе (bio)
  4. Интересы (выбрать минимум 3)
  5. Параметры поиска
  
- [ ] **Прогресс индикатор:**
  - Показывать шаг N из 5
  - Можно вернуться назад
  
- [ ] **Анимации между шагами:**
  - Плавные переходы
  - Конфетти на последнем шаге («Готово!»)
